학급별 주간 시간표

  1. 반 개수와 최대 교시를 먼저 정하고, 전담 과목에서 필요한 과목(없으면 A~F/강사1~3)을 체크하세요.
  2. 자동 생성 전에는 체육을 먼저 배치하고, 마지막 교시 X 설정으로 요일별 마지막 교시 사용 여부를 정하세요.
  3. 학급 시간표 칸에 과목을 직접 입력하거나, 전담 시간표 칸에 반 번호를 입력해도 같은 위치에 서로 연동됩니다.
  4. 자동 생성은 설정한 시수대로 전담 과목을 배치하며, 다시 실행해도 수동 확정값/체육/X 설정은 유지됩니다.
  5. 중복 확인 버튼으로 충돌 여부를 빠르게 점검하세요.
